# CLI UI安装

作为san-cli的可视化命令，安装san-cli即可尽情享用。

全局安装san-cli

# use npm 
> npm install -g san-cli

# or use yarn 
> yarn global add san-cli

之后执行命令启动

> san ui

这样就会打开一个浏览器窗口，通过图形化界面安装和管理项目了。
